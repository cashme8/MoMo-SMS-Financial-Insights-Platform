## Project Overview
This project processes Mobile Money (MoMo) SMS data provided in XML format.
We extract, clean, categorize, and store transaction data in a relational database,
then visualize insights using a web-based dashboard.

The system demonstrates enterprise-level full-stack design, ETL pipelines,
and Agile team collaboration.
